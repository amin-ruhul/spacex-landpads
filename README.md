## Setup (run locally)

```bash
# clone the repository
  git clone git@github.com:amin-ruhul/spacex-landpads.git
  cd spacex-landpads
```

```bash
  # copy the .env.example file to .env and set the environment variables.
  cp .env.example .env
```

```bash
  # install dependencies
  npm install
```

```bash
  # run the app
  npm run dev
```
